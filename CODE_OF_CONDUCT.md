# Code of Conduct

This project is public-facing and brand-adjacent. Keep interactions respectful, practical, and artifact-focused.

## Expected behavior
- Be clear and specific
- Critique work, not people
- Assume good faith unless there is strong evidence otherwise
- Keep feedback useful and proportionate

## Unacceptable behavior
- personal attacks
- harassment
- bad-faith pile-ons
- reposting or misrepresenting private context as public canon

## Scope
This applies to repository discussions, issues, pull requests, and direct contribution exchanges related to this project.
