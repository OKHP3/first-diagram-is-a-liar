# Contributing

Thanks for your interest in **The First Diagram Is Usually a Liar**.

This repository is primarily a public artifact archive and methodology record, not a conventional open-source software package.

## Helpful contributions
- flagging broken links
- identifying mislabeled diagrams or assets
- suggesting documentation clarifications
- proposing cleaner artifact organization

## Please avoid
- large unsolicited editorial rewrites of the article
- re-scoring the council record as if unofficial changes are canonical
- restructuring the repo without first explaining the problem being solved

## How to contribute
1. Be specific about the file, page, or artifact in question.
2. Describe the problem first, then the proposed improvement.
3. Keep suggestions practical and artifact-focused.
