# Security Policy

This repository is an artifact archive and methodology project, not a production software service.

## Reporting concerns

If you discover:
- exposed credentials
- unintended private content
- broken or misleading public links
- any other repository security concern

please report it privately to:

**contact@overkillhill.com**

## Scope

This repo does not currently include a public bug bounty program.

## Response intent

Reasonable reports will be reviewed and, when valid, corrected as quickly as practical.
