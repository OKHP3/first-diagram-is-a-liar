# Article Drafts

This folder holds article drafts and working prose for **"The First Diagram Is Usually a Liar."**

Files are versioned by milestone:

- `v0.1.md` — protoform draft: thesis stated, argument rough, structure loose
- Later versions expand the argument, incorporate diagram contest outcomes, poll results, and council synthesis

The goal is to preserve lineage. Every meaningful revision gets its own file rather than overwriting the previous one. The progression from protoform to published article is part of the story.

When a version is considered stable enough for external review or LinkedIn publication, it gets noted in `/archive/changelog.md`.

**Do not delete earlier versions.** The evolution of the argument is the point.
