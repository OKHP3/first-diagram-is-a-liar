# Captions and Alt Text

This file provides guidance for alt text and captioning for all public images in the ETCH-AI-SKETCH project.

## General guidance
- Include model name, version, and round.
- Provide context: what concept, method, or insight the image represents.
- Keep text concise and readable.

## Examples
- `Copilot V1`: Lane-based diagram showing the early Round 1 layout; highlights workflow loops and corrections.
- `Claude V2`: Synthesized V2 diagram illustrating iterative improvements; emphasizes feedback loops and structural clarity.
- `Notion V1`: Example layout showing a structured comparison with V1 Copilot.
- `Replit V2`: Demonstrates layout variations and key decision forks for Round 2.

## Slide decks
- Include title slide and annotate each slide with purpose and key takeaway.

## Mermaid diagram links
- For each image, include public link to original Mermaid source to allow exploration in high resolution.
