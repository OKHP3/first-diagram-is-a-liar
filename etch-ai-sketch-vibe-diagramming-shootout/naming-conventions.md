# Naming Conventions

This document defines the public-facing naming pattern for ETCH-AI-SKETCH assets.

## Diagram sources
Format:
`etch-ai-sketch-using-a-council-to-design-at-velocity-{model}-v{round}.mmd`

Examples:
- `etch-ai-sketch-using-a-council-to-design-at-velocity-copilot-v1.mmd`
- `etch-ai-sketch-using-a-council-to-design-at-velocity-claude-v2.mmd`

## Image exports
Formats:
- high-resolution: `...-{model}-v{round}.png`
- low-resolution: `...-{model}-v{round}-lores.png`

## Slide decks
Formats:
- square presentation: `...-square.pptx` / `...-square.pdf`
- wide presentation: `...-wide.pptx` / `...-wide.pdf`

## Special variants
When an entry is materially different from the normal round path, append a qualifier.
Preferred format:
- `ChatGPT V2 Pro`
- `...-chatgpt-v2pro.*`

If later normalization is desired, a future rename to `v2-pro` is acceptable as long as all links and references are updated together.

## Public-facing labels
Use human-readable forms on the website and in the article:
- `Copilot V1`
- `Claude V2`
- `ChatGPT V2 Pro`
- `Notion V1`
- `Replit V2`
