# Images

This folder holds cover images, article illustrations, social media assets, and contest winner visuals for **"The First Diagram Is Usually a Liar."**

## Folder Structure

```
/covers/     Hero and header images for the article and website
/article/    Inline images used within the article itself
/social/     Cropped or formatted versions for LinkedIn, Twitter/X, etc.
/winners/    Final selected diagram images from contest rounds
```

## Usage Notes

- Export covers at 1920×1080 minimum for website use
- Social images should be sized for LinkedIn (1200×627 recommended)
- Winners should be stored at full resolution; thumbnails go in `/diagrams/thumbnails/`

## Naming Convention

Use descriptive lowercase filenames with hyphens. Include context: `cover-v1-dark.png`, `social-poll-round-a.png`, `winner-v2-council-pick.png`.
