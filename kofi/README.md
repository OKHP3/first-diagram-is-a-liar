# Ko-fi

This folder holds companion product copy, supporter bundle ideas, and monetization notes for **"The First Diagram Is Usually a Liar."**

## Purpose

Ko-fi is the low-pressure mechanism for turning the public work in this repo into something supporters can back directly. It is not a paywall. The core article and all public materials remain free. This is for people who want to go deeper or who want to support the project directly.

## Files

- `product-copy.md` — Copy for Ko-fi product listings (bundles, downloads, notes)
- `bundle-notes.md` — Ideas and planning for supporter bundles
- `membership-ideas.md` — Notes on potential recurring membership tiers

## Tone

Practical. Not hype-driven. Do not oversell. These are companion materials for people who found the public work useful and want more.

If something is not worth paying for, it should not be listed. If something is worth paying for, describe it clearly and let it speak for itself.
