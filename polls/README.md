# Polls

This folder holds LinkedIn poll planning materials, option labels, option writeups, supporting comment copy, and engagement strategy for the **"The First Diagram Is Usually a Liar"** publication sequence.

## Files

- `poll-copy.md` — Draft poll question text and answer options for each round
- `option-bios.md` — Explanatory writeups for each poll option (used in comments)
- `release-plan.md` — Publication sequence and milestone schedule

## Purpose

Polls are not decoration. They are part of the argument.

Each poll round serves a function: generating signal about which diagrams communicate better, which arguments land, and which framings resonate. The results feed back into article revisions and council synthesis.

Plan the polls before launching them. Write the supporting comment copy in advance. Know what you are measuring before you post.
