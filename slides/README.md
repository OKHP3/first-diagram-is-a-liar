# Slides

This folder holds ETCH-AI-SKETCH slide decks, PDF exports, and presentation notes supporting the article **"The First Diagram Is Usually a Liar."**

## Folder Structure

```
/pdf/      Exported PDF versions of decks
/pptx/     Source PowerPoint or equivalent deck files
/notes/    Speaker notes, narrative outlines, and deck rationale
```

## ETCH-AI-SKETCH Decks

ETCH-AI-SKETCH is the working name for the presentation format used in this project: structured decks that pair diagram visuals with the article argument, optimized for LinkedIn document posts.

Decks should be clean, readable at thumbnail size, and consistent with the graphite/industrial tone of the project.

## File Conventions

- Name deck files by topic and version: `diagram-liar-intro-v1.pdf`
- Keep source files in `/pptx/` even if Google Slides is used for editing (export to PPTX for archiving)
- Store speaker notes and narrative context in `/notes/` as plain markdown files
