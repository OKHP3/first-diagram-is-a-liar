# Website

This folder holds static HTML content and assets for the public-facing hub for **"The First Diagram Is Usually a Liar."**

## Structure

```
index.html         Main entry point
/assets/css/       Stylesheets
/assets/img/       Images used on the site
```

## Purpose

This is a GitHub Pages-compatible static site. No build tools. No framework. Plain HTML and CSS.

The site functions as a hub: linking to the article, diagrams, slides, polls, and council notes. It does not duplicate content — it surfaces and connects it.

## Deployment

Enable GitHub Pages in the repository settings and point it at the `/web/` folder (or the root, if preferred). The `index.html` file is the entry point.

## Design Notes

Keep it clean. Dark/graphite tone. Semantic HTML. Readable at arm's length. No animations, no carousels, no unnecessary JavaScript.

The site should feel like a serious working project, not a product landing page.
